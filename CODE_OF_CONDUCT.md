# Code of Conduct

This project expects direct, respectful technical collaboration.

## Expected Behavior

- Be clear, factual, and constructive.
- Critique code and decisions, not people.
- Respect maintainer scope decisions.
- Avoid harassment, personal attacks, or discriminatory language.

## Reporting

If conduct becomes a problem, open a minimal issue asking maintainers for a private contact path. Do not include sensitive personal information in public issues.

Maintainers may remove comments, close threads, or block participants who make collaboration unsafe or unproductive.
