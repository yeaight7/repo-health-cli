# Security Policy

## Supported Versions

`repo-health-cli` is pre-1.0. Security fixes target the current `main` branch until versioned releases begin.

## Reporting a Vulnerability

Use GitHub private vulnerability reporting if it is enabled for this repository. If private reporting is unavailable, open a minimal public issue asking for a secure contact path and do not include exploit details.

Please include:

- affected version or commit
- reproduction steps
- expected impact
- any safe mitigation ideas

The project does not use paid APIs or external runtime services in local audit mode.
